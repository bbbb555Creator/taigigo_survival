## 🎮 ゲームの特徴

- **4つの難易度階層**: 
    - 🟢 **初級**: 日常的な基本語彙
    - 🟡 **中級**: 社会生活で使われる語彙
    - 🔴 **上級**: 抽象概念や難読熟語
    - 🔥 **地獄級**: 業界用語・専門知識・マニアックな対比
- **全国ランキング**: スコアを登録して、全国の猛者と競い合おう！
- **PWA対応**: スマホのホーム画面に追加すれば、いつでもすぐプレイ可能。
